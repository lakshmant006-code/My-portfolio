Logo — the Hiku brand lockup (green mountain mark + optional wordmark).

```jsx
<Logo height={30} />
<Logo height={24} wordmark />
```

`src` points at the mountain PNG (default `assets/logo-mark.png` — adjust the relative path to where it's used). Set `wordmark` to add the "Hiku" text in Satoshi Black. There is no vector logo source, so the mark is always the raster PNG.
