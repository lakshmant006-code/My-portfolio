Illustration — flat, monochrome hiking scenes in the Hiku green palette.

```jsx
<Illustration scene="mountains" height={160} />
<Illustration scene="tent" tone="deep" />
<Illustration scene="forest" tone="soft" height={120} />
```

Scenes: `mountains`, `tent`, `forest`, `trail`, `campfire`, `lake`. `tone` picks a green tonal ramp (`green` · `deep` · `soft`). Every scene is single-hue (monochrome green) — use for empty states, onboarding, community/section headers, and as the graceful fallback behind real photography.
