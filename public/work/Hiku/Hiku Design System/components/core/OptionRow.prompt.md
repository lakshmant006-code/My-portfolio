OptionRow — a bordered, tappable list row with icon, title and subtitle.

```jsx
<OptionRow icon="caravan" title="RV" subtitle="find places that fit your rig" />
<OptionRow icon="tent" title="Tent" subtitle="Pitch your own tent" selected />
<OptionRow icon="house" title="Lodging" subtitle="Accommodations provided" />
```

Selected state adds a green border + ring. Stack rows in a column with `gap: 12px`.
