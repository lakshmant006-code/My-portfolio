IconButton — icon-only control for toolbars, headers and map overlays.

```jsx
<IconButton icon="sliders-horizontal" ariaLabel="Filters" />
<IconButton icon="layers" shape="round" variant="float" ariaLabel="Map layers" />
<IconButton icon="arrow-left" variant="plain" ariaLabel="Back" />
```

`shape`: square (rounded rect, header) or round (map FABs). `variant`: outline · float (drop-shadow FAB) · plain · brand. `active` tints border+icon green.
