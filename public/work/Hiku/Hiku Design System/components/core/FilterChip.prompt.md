FilterChip — compact pill filter shown in a row under the search bar.

```jsx
<FilterChip dropdown>Camping style</FilterChip>
<FilterChip dropdown>Amenities</FilterChip>
<FilterChip selected>Pets allowed</FilterChip>
```

`dropdown` adds a chevron; `selected` fills soft green (`--brand-wash`) with a green border. Lay several out in a horizontal flex row with `gap: 8px`.
